The Laplace transform of a function $f(t)$, defined for all real numbers $t \geq 0$ is the function $F(s)$, which is a unilateral transform defined by:
\begin{equation*}
    F(s) = \int_0^\infty f(t)e^{-st} dt
\end{equation*}
